# graphql-client-ex
GraphQL Client Sample Project

> NOTE: 본 프로젝트는 GraphQL Client만 포함하고 있는 저장소이기에, Server 저장소인 graphql-ex와 함께 실행햐아 합니다.

# 개발환경
- node.js v12
- Vue.js v2.6
- yarn v1.21

# 실행방법
1. 저장소 Clone
2. 저장소 안에서 의존성 설치
   ```bash
   $ yarn

   # or you can use npm
   $ npm i
   ```
3. 실행
   ```bash
   $ yarn serve

   # or you can use npm
   $ npm run serve
   ```
